## Technical Requirements for Go-Live

### Approvals Needed
- [ ] **Board/Management** — Approve production deployment (live payment processing)
- [ ] **Finance** — Confirm £899 pricing is correct
- [ ] **Legal** — Confirm T&Cs and payment terms are acceptable

### Technical Setup Required

**Stripe (10 minutes)**
- Create live webhook at: `https://ai-confidence-day.vercel.app/api/webhook`
- Copy live secret key: `sk_live_...`
- Copy webhook signing secret: `whsec_...`

**Resend (15-30 minutes)**
- Verify domain: `solvedtogether.co.uk`
- Add DNS records (CNAME/TXT provided by Resend)
- Wait for verification completion

**HubSpot (5 minutes)**
- Verify token has scopes: `crm.objects.deals.write` + `crm.objects.notes.write`
- Regenerate token if scopes missing

**Vercel (5 minutes)**
- Update environment variables in production:
  - `STRIPE_SECRET_KEY` → live key
  - `STRIPE_WEBHOOK_SECRET` → webhook signing secret

**Code (5 minutes)**
- Update checkout amount: `/api/create-checkout.js` line 40
  - Change: `unit_amount: 100` → `unit_amount: 89900`
- Deploy to Vercel (auto-deploys on git push)
